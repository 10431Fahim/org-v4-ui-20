# Social Media Meta Tags Test

## Test your social media sharing with these tools:

### 1. Facebook Sharing Debugger
- URL: https://developers.facebook.com/tools/debug/
- Paste your founding-historic page URL
- Click "Debug" to see how Facebook will display your link

### 2. Twitter Card Validator
- URL: https://cards-dev.twitter.com/validator
- Enter your page URL
- Check if Twitter Card is working properly

### 3. LinkedIn Post Inspector
- URL: https://www.linkedin.com/post-inspector/
- Enter your page URL
- See how LinkedIn will display your content

### 4. WhatsApp Link Preview
- Send your page URL in WhatsApp
- Check if image and description appear correctly

### 5. Telegram Link Preview
- Send your page URL in Telegram
- Verify the preview shows properly

## Expected Results:

When sharing your founding-historic page, you should see:

✅ **Facebook/LinkedIn**: Large image preview with title and description
✅ **Twitter**: Summary card with large image
✅ **WhatsApp/Telegram**: Image preview with title and description
✅ **All platforms**: Proper BNP branding and professional appearance

## Troubleshooting:

If images don't appear:
1. Check if image URLs are absolute (https://bnpbd.org/...)
2. Verify images are publicly accessible
3. Ensure images are at least 1200x630 pixels
4. Test with social media debuggers
5. Check server logs for crawler requests
